## 4. 后端API

| 端点 | 说明 |
|------|------|
| `GET /overview` | KPI + 分布 + 异动 |
| `GET /overview/tag-changes` | 标签变化 |
| `GET /overview/scatter-data` | 散点图 |
| `GET /customer/{id}` | 详情 |
| `GET /customer/{id}/history` | 分析历史 |
| `POST /alerts/{id}/acknowledge` | 确认异动 |
