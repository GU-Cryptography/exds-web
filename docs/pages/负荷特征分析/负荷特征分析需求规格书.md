# 负荷特征分析与异动监控模块需求规格说明书

## 1. 概述 (Overview)

本模块旨在通过对用户电力负荷数据的深度挖掘，自动生成能够反映企业经营趋势、生产规律、宏观风险及用电偏好的特征标签。同时，建立基于滑动窗口的异动监控机制，及时发现客户生产模式的异常变化（如停产、扩产、排班调整、私接光伏等），为交易决策和风控提供数据支撑。

## 2. 负荷特征标签体系 (Tagging System)

标签体系分为基于长周期（一年）数据的“宏观经营标签”和基于短周期（近30天）数据的“微观生产标签”。

### 2.1 长周期：经营与行为特征 (Yearly / Long-term)

**数据源**：近一年日电量数据（X轴为日期，Y轴为日电量）。

#### 2.1.1 经营趋势类 (Business Trend)
| 标签名称 | 标识 | 判定逻辑 | 业务价值 |
| :--- | :--- | :--- | :--- |
| **扩产爬坡型** | 🚀 | 近3个月日均电量环比增长 > X%，且趋势线斜率 > Y | 需调增合同，防超用。 |
| **减产萎缩型** | 📉 | 近3个月日均电量环比下跌 > X%，且趋势线斜率 < -Y | 需调减合同，防不足考核；关注回款。 |
| **稳健经营型** | ⚓ | 日均电量年度变动率在 ±Z% 以内 | 优质压舱石客户。 |

#### 2.1.2 生产日历类 (Calendar Pattern)
| 标签名称 | 标识 | 判定逻辑 | 业务价值 |
| :--- | :--- | :--- | :--- |
| **标准双休型** | 📅 | 周末电量显著低于工作日（具体阈值待定） | 周末需下调申报。 |
| **周末生产型** | 🏭 | 周末与工作日无差异 | 预测无须区分周末。 |
| **春节深调型** | 🧨 | 春节期间电量 < 平时 N%，持续 > M 天 | 1-2月重点关注复工。 |
| **节后慢热型** | 🐌 | 节后第1周日均电量 < 平时 K% | 节后预测需保守。 |

#### 2.1.3 稳定性与风险类 (Risk & Stability)
| 标签名称 | 标识 | 判定逻辑 | 业务价值 |
| :--- | :--- | :--- | :--- |
| **剧烈波动型** | 🌊 | 日电量变异系数 (CV) > 阈值A | **高危**，偏差考核重灾区。 |
| **间歇停产型** | 🛑 | 全年零电量/极低电量天数占比 > 阈值B | 经营不稳定，需核实。 |
| **极度规律型** | 📏 | 日电量变异系数 (CV) < 阈值C | 优质基准负荷。 |

#### 2.1.4 季节气象类 (Seasonal)
| 标签名称 | 标识 | 判定逻辑 | 业务价值 |
| :--- | :--- | :--- | :--- |
| **冬夏双峰型** | ❄️🔥 | 7月或1月日均电量 >春秋月份 N倍 | 关注气温，备战迎峰度夏/冬。 |
| **气温敏感型** | 🌡️ | 日电量与气温相关系数 > R1 | 预测需引入气温变量。 |
| **气温钝化型** | 🧱 | 日电量与气温相关系数 < R2 | 预测可忽略气温。 |

> **注意**：季节性标签需单独调用长周期数据计算，即使在短周期视图中也应保留。

### 2.2 短周期：生产与用电特征 (Monthly / Short-term)

**数据源**：近30天96点/48点负荷曲线数据。
**理由**：近一个月数据最能代表当前生产状态，时效性强，计算性价比高。

#### 2.2.1 生产班次类 (Production Shift)
| 标签名称 | 标识 | 曲线形态 | 判定逻辑 (基于月度均值曲线) |
| :--- | :--- | :--- | :--- |
| **连续生产型** | 🏭 | “一”字形 | 平均负荷率 > X%，Min/Max > Y% |
| **白班单班型** | ☀️ | “凸”字形 | 08:00-18:00 电量占比 > Z%，夜间负荷极低 |
| **双班/三班型** | 🌙 | “几”字形 | 高负荷持续时长 > 16小时 |
| **夜间生产型** | 🦉 | “凹”字形 | 谷时段(22:00-08:00)电量占比 > K% |

#### 2.2.2 行为规律类 (Behavior Pattern)
| 标签名称 | 标识 | 形态特征 | 判定逻辑 (30天样本统计) |
| :--- | :--- | :--- | :--- |
| **周末双休型** | 📅 | “5高2低” | 周末曲线大幅坍塌 |
| **机器规律型** | 🤖 | 千篇一律 | 日曲线间平均余弦相似度 > 0.95 |
| **随机波动型** | 🎲 | 杂乱无章 | 日曲线相似度低，变异系数高 |

#### 2.2.3 能源设施类 (Energy Asset)
| 标签名称 | 标识 | 形态特征 | 判定逻辑 |
| :--- | :--- | :--- | :--- |
| **光伏自备型** | 🔋 | “马鞍”形 | 中午(11-14点)负荷显著凹陷，凹陷深度与晴天相关 |
| **储能套利型** | ⚡ | 削峰填谷 | 深夜突增(充电)，尖峰骤降(放电) |

#### 2.2.4 成本偏好类 (Cost Sensitivity)
| 标签名称 | 标识 | 含义 | 判定逻辑 |
| :--- | :--- | :--- | :--- |
| **成本敏感型** | 💰 | 避峰用电 | 尖峰时段电量占比显著低于行业平均 |
| **刚性用电型** | 💸 | 不对价格敏感 | 尖峰时段负荷无明显避让 |

> **UI提示**：短周期标签旁应标注“基于近30天数据”。

## 3. 异动监控机制 (Anomaly Detection)

核心逻辑：**滑动窗口对比 (Sliding Window)**。
- **基准窗口 (Baseline)**：过去30天平均曲线。
- **观测窗口 (Observation)**：最近3-5天平均曲线。
- **判定**：计算两窗口特征差异，超过阈值即报警。

### 3.1 四大核心异动特征

#### 1. 形状异动 (Shape Drift) - 最根本变化
- **含义**：生产模式质变（如换班次、加光伏）。
- **指标**：**余弦相似度 (Cosine Similarity)** < 0.85。
- **场景**：连续转单班、光伏接入。

#### 2. 重心异动 (Peak Shift) - 时段转移
- **含义**：调整生产时间，错峰。
- **指标 1**：**峰值时刻偏移 (Peak Time Shift)** > N小时。
- **指标 2**：**昼夜用电比 (Day/Night Ratio)** 变化幅度 > M%。

#### 3. 力度异动 (Scale Drift) - 量级变化
- **含义**：扩产或停产。
- **指标 1**：**负荷率偏离 (Load Factor Deviation)** > K%。
- **指标 2**：**容量利用率 (Capacity Drop)** 连续3天 < 10% (疑似停产)。

#### 4. 规律异动 (Stability Decay) - 预测失效
- **含义**：生产变得不可预测。
- **指标**：**波动率 (Volatility Spike)**。最近5天标准差 > 过去30天平均标准差 * 2。

## 4. 界面功能设计 (UI/UX)

### 4.0 功能目标

本模块的UI设计核心服务于以下三大业务目标：

1.  **服务负荷预测**：通过特征画像，量化无规律客户的数量和电量权重，为预测模型提供输入，识别预测难点。
2.  **及时掌握异动**：通过异动监控雷达，第一时间捕捉客户生产模式变化，动态修正预测模型，提高预测准确度。
3.  **客户优劣评价**：通过规律性评分、标签组合，对客户进行分级评价，识别优质"压舱石"客户和高风险"波动源"客户。

#### 4.1.2 A. 顶部 KPIs 面板

**设计目标**：一眼掌握代理客户整体画像覆盖率、结构健康度及当日风险。

#### 4.1.3 B-1. 中部左侧：异动告警信息 (Anomaly Radar)

**设计目标**：今日必须处理的待办事项列表，按风险排序。

##### 控制栏
- **监测窗口选择**：`近3天 vs 近30天基准` (默认) | `近7天 vs 近30天`
- **异动类型筛选**：`[全部]` `[⚠️ 形状突变]` `[📉 规模突变]` `[🕒 时间漂移]` `[🌊 波动加剧]`

##### 异动报警列表 (Actionable List)

| 列名       | 内容                                                                 |
| :--------- | :------------------------------------------------------------------- |
| 风险等级   | 🔴 高危 / 🟠 中风险 / 🟡 低风险                                      |
| 客户名称   | 加粗显示，带行业图标                                                 |
| 异动类型   | `形状突变` / `规模突变` / `时间漂移` / `波动加剧`                     |
| 异动描述   | 机器诊断标签 (e.g., "相似度跌至 0.65，**疑似光伏介入**")              |
| 发生时间   | `近3天` / `昨日`                                                     |
| 操作       | `[查看对比]` `[忽略]` `[已确认]`                                     |

##### 异动对比弹窗 (点击 `[查看对比]` 触发)
- **图表展示**：
    - **深色实线**：最近N天平均曲线（现状）
    - **浅色虚线/阴影**：过去30天平均曲线（基准）
    - **红色高亮区域**：两者差异最大的时段
- **归因诊断标签**：系统根据差异特征自动推断原因
    - *"中午负荷显著凹陷 → 疑似光伏介入"*
    - *"夜间负荷归零 → 疑似夜班取消"*
- **操作按钮**：`[确认归因]` `[线下核实]` `[忽略本次]`

#### 4.1.4 B-2. 中部右侧：特征分布总览 (Feature Atlas)

**设计目标**：了解代理客户群体的"成分构成"。

##### 特征标签分布图
- **图表类型**：**环形图 (Donut Chart)** 或 **分组柱状图 (Grouped Bar Chart)**
- **维度**：
    - **按生产班次**：连续(30%)、单班(40%)、双班(20%)、其他(10%)
    - **按特殊设施**：无(80%)、光伏自备(15%)、储能(5%)
- **交互**：点击扇区/柱子，下方客户列表自动筛选为该分类。

##### 规模-稳定性矩阵 (可选高级视图)
- **图表类型**：**散点图 (Scatter Plot)**
- **X轴**：日均电量（规模）
- **Y轴**：波动率 CV（稳定性，越高越不稳定）
- **点大小**：电量权重
- **区域划分**：
    - **右上角（大而乱）**：🔴 重点治理对象（考核风险源）
    - **右下角（大而稳）**：🟢 核心压舱石客户
    - **左下角（小而稳）**：🟢 长尾优质客户
- **交互**：框选区域可批量筛选客户列表。

#### 4.2.3 A. 长周期分析模块

**数据源**：近一年日电量数据。

##### A1. 日电量趋势曲线

- **图表类型**：**折线图 + 面积图**
- **X轴**：日期 (近1年)
- **Y轴**：日电量 (kWh)
- **辅助线**：
    - **月均值参考线**：虚线标注每月均值。
    - **趋势线 (可选)**：线性回归趋势线，判断扩产/减产。
- **交互**：
    - 支持时间范围缩放 (近3月/6月/1年)。
    - 悬停显示日期、电量、同比/环比。
- **特殊标注**：
    - **红色竖线**：标注已知停产/检修日期。
    - **绿色竖线**：标注节假日 (春节、国庆等)。

##### A2. 月电量柱状图

- **图表类型**：**柱状图**
- **X轴**：月份 (近12月)
- **Y轴**：月累计电量 (kWh)
- **辅助信息**：柱子上方标注环比增长率 (e.g., `+5%` 绿色, `-10%` 红色)。
- **交互**：点击柱子，A1区域自动聚焦至该月日电量视图。

##### A3. 长周期特征文字评价

> **设计目标**：用自然语言总结长周期数据特征，辅助交易员快速理解客户。

**评价模板 (由系统自动生成)**：

```
------------------------------------------------------------------
📊 长周期特征评价 (基于近1年日电量数据)
------------------------------------------------------------------

【经营趋势】
该客户近一年日均电量为 **X kWh**，年度累计电量 **Y MWh**。
近3个月日均电量环比 **增长/下降 Z%**，趋势线斜率为 **+/-K**。
▶️ 判定标签：[⚓ 稳健经营型] / [🚀 扩产爬坡型] / [📉 减产萎缩型]
```

#### 4.2.4 B. 短周期分析模块

**数据源**：近30天48点/96点负荷曲线数据。

##### B1. 短周期均值曲线

- **图表类型**：**折线图 (双线对比)**
- **主线 (实线)**：近30天所有日曲线的**均值曲线** (代表典型日形态)。
- **对比线 (虚线/阴影，可选)**：
    - 工作日均值 vs 周末均值。
    - 或历史基准期 (如上月) 均值。
- **X轴**：时刻 (00:00 - 23:30, 48点)
- **Y轴**：负荷 (kW)
- **辅助标注**：
    - 标注峰值时刻、谷值时刻。
    - 分时电价时段背景色 (尖/峰/平/谷)。

##### B2. 短周期波动率展示

- **图表类型**：**折线图 + 置信带 (Confidence Band)** 或 **热力图 (Heatmap)**
- **方案一：带状图**
    - 展示均值曲线，外围用阴影表示 ±1 标准差波动范围。
    - 阴影越宽，表示该时段波动越大。
- **方案二：热力图**
    - X轴：日期 (近30天)
    - Y轴：时刻 (00:00-23:30)
    - 颜色深浅：负荷高低。
    - 直观展示每日曲线的一致性或差异。
- **核心指标展示**：
    - **曲线相似度**：近30天日曲线间平均余弦相似度 (e.g., `0.92`)。
    - **变异系数 (CV)**：日内负荷波动率 (e.g., `15%`)。

## 5. 数据处理与后台需求

### 5.1 批处理任务 (Batch Jobs)
- **频率**：每日凌晨执行。
- **任务内容**：
    1. **全量特征更新**：计算近30天/1年指标，更新 Tags。
    2. **异动扫描**：计算 Sliding Window 指标，生成 Alert 记录。

### 5.2 核心计算指标
需计算并存储以下指标的**当前值**与**变化率**：
1. **Cosine_Similarity** (形状)
2. **Peak_Time** & **Peak_Valley_Ratio** (时段/成本)
3. **Avg_Load_Rate** & **Capacity_Usage** (规模)
4. **Daily_CV** (规律性)
