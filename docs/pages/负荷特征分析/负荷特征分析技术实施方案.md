## 1. 概述

本文档基于 [需求规格说明书](file:///d:/Gitworks/exds-web/docs/requirements/load_characteristic_analysis.md)，结合现有项目代码结构、API模式和数据库设计，制定详细的技术实施方案。

## 2. 现有架构分析

### 2.1 后端架构

| 层级 | 位置 | 说明 |
| :--- | :--- | :--- |
| **API 路由** | `webapp/api/v1_*.py` | 模块化路由文件，通过 `router.include_router()` 注册 |
| **服务层** | `webapp/services/*.py` | 业务逻辑，如 `LoadQueryService`, `CustomerService` |
| **数据模型** | `webapp/models/*.py` | Pydantic 模型定义 |
| **数据库** | `webapp/tools/mongo.py` | MongoDB 连接，全局 `DATABASE` 实例 |

**可复用的核心服务**:
- `LoadQueryService`: 提供 `get_curve_series()`, `get_daily_totals()`, `get_monthly_totals()`, `batch_get_*` 方法。
- `CustomerService`: 客户 CRUD，标签管理 (`add_tag`, `remove_tag`)。
- `CustomerLoadOverviewService`: Dashboard KPI/图表/列表数据，可作为参考模式。

### 2.2 前端架构

| 层级 | 位置 | 说明 |
| :--- | :--- | :--- |
| **API 客户端** | `frontend/src/api/*.ts` | Axios 封装，如 `customerLoadOverview.ts` |
| **页面组件** | `frontend/src/pages/*.tsx` | 页面级组件 |
| **通用组件** | `frontend/src/components/*.tsx` | 可复用组件 |
| **Hooks** | `frontend/src/hooks/*.tsx` | 如 `useChartFullscreen`, `useSelectableSeries` |
| **路由** | `frontend/src/App.tsx` | 移动端路由定义 |

**可参考的页面模式**: `CustomerLoadOverviewPage.tsx` (KPI + 图表 + 列表三段式布局)。

### 2.3 数据库结构

| 集合 | 用途 |
| :--- | :--- |
| `unified_load_curve` | 客户日负荷曲线 (48点 `mp_load.values`, `meter_load.values`, `tou_usage`) |
| `customer_archives` | 客户档案，**含 `tags` 数组** |
| `customer_tags` | 标签定义 (类别、来源、规则描述) |

**新增集合** (本模块需要):
- `customer_characteristics`: 存储计算后的特征指标和标签快照。
- `customer_anomalies`: 存储异动告警记录和处理状态。

定时任务：每日客户特征更新与异动扫描
使用 cron 或 APScheduler 调度，建议 02:00 执行

from webapp.services.characteristic_service import CharacteristicService
from webapp.services.anomaly_service import AnomalyService

def main():
    char_service = CharacteristicService()
    anomaly_service = AnomalyService()
    
    # Step 1: 更新所有客户特征
    char_result = char_service.batch_update_all_customers()
    print(f"特征更新完成: {char_result}")

    anomaly_result = anomaly_service.batch_scan_all_customers()
    print(f"异动扫描完成: {anomaly_result}")

if __name__ == "__main__":

import apiClient from './client';

export interface CharacteristicsOverview {

  regularity_score: number;
