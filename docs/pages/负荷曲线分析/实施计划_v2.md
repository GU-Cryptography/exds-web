# 整体负荷分析页面重构

## 需求背景

将 `LoadAnalysisPage.tsx` 重构为**单页面布局**，聚焦于**所有客户负荷汇总后的宏观分析**。

---

## 已确认需求

| 项目 | 确认内容 |
|------|----------|
| 数据粒度 | **30分钟电量值**（48点/日） |
| 年份范围 | 固定 **2025 + 2026** 两年 |
| 节假日数据 | 使用 `china_holidays_manual.py` |
| 对比逻辑 | 昨日 / 7天前 / 同月同日 |
| 统计重点 | **时段电量分配**（非负荷指标） |
| 峰谷比 | = (尖峰 + 高峰) / (低谷 + 深谷) |
| 移动端 | 单列四区块布局 |

---

## 页面布局

### 桌面端（两列）
```
┌────────────────────────────────────┬────────────────────────────────────────┐
│ 【月度电量柱状图】                  │ 【日电量柱状图】 ◀ [上月] [下月]       │
│ 2025灰色 / 2026蓝色               │ 工作日/周末/节假日颜色区分              │
│ tooltip显示同比                   │ 均值参考线                             │
├────────────────────────────────────┴────────────────────────────────────────┤
│ 【日内电量曲线】                        【统计面板】                         │
│ 日期选择器 + 对比选项                   [当日] [当月] [年度] ← Tab切换       │
│ 支持多日对比 / 工作日均值               时段分配表格 + 条形图                 │
│ TOU背景色分区                          峰谷比(颜色指示)                      │
└─────────────────────────────────────────────────────────────────────────────┘
```

---

## 模块需求

### 模块1：月度电量柱状图

- 2025年浅灰色，2026年蓝色渐变
- 当月未完整显示虚线边框
- **Tooltip显示同比**：`2026年1月: 120万kWh (+5.2%)`
- 点击月份联动模块2

### 模块2：日电量柱状图

- 工作日/周末/节假日颜色区分
- **均值参考线**（虚线）
- **上月/下月快捷按钮**
- 点击日期联动模块3

### 模块3：日内电量曲线

- 独立日期选择器
- 对比选项：昨日 / 上周 / 去年
- **支持多日对比**（勾选最近7天）
- **工作日均值**作为对比基准
- TOU背景色分区
- 横屏全屏支持

### 模块4：统计面板

- **Tab切换**：当日 / 当月 / 年度
- 时段电量分配表格 + **水平条形图**
- **峰谷比颜色指示**：<1.5绿 / 1.5~2.0橙 / >2.0红
- 同比变化百分比
