# 分时电价时段划分页面 UI 设计方案 (最终版)

## 设计目标
通过“全局视角”让用户一眼看清全年时段分布规律，同时明确展示各时段的电价系数关系。采用响应式设计，根据**屏幕方向**动态切换可视化图表。

## 1. 核心配色标准
*   🔴 **尖峰 (Critical Peak)**: `error.main` (#d32f2f)
*   🟠 **高峰 (Peak)**: `warning.main` (#ed6c02)
*   🔵 **平段 (Flat)**: `info.main` (#0288d1)
*   🟢 **低谷 (Valley)**: `success.main` (#2e7d32)
*   🟣 **深谷 (Deep Valley)**: `secondary.main` (#9c27b0)

## 2. 横屏模式 UI (Landscape View)

**适用场景**: 桌面端, 平板, **手机横屏**

**核心策略**: **横向热力图** (X轴=时间)。利用宽屏优势展示完整 24 小时轴。

```text
+-----------------------------------------------------------------------+
| [页面标题] (手机横屏时隐藏)                                           |
+-----------------------------------------------------------------------+
| [控制栏] 政策版本: [2025-07-01 ▼]                                     |
|          图例: [🔴 尖峰] [🟠 高峰] [🔵 平段] [🟢 低谷] [🟣 深谷]        |
+-----------------------------------------------------------------------+
|                                                                       |
|  [左侧: 全年横向热力图] (SVG绘制, X轴=时间, Y轴=月份组)               |
|  +--------------------------------------------+                       |
|  | 1月  [====谷====----平----++++峰++++]      |                       |
|  | ...                                        |                       |
|  | 夏   [====谷====++++++++++峰++++++++]      |                       |
|  | ...                                        |                       |
|  | 12月 [====谷====----平----++++峰++++]      |                       |
|  +--------------------------------------------+                       |
|    00:00       12:00       24:00 (X轴)                                |
|                                                                       |
|  [下方注释信息]                                                       |
|  1. **价格系数说明**：(例如：当前版本 2025-07-01 生效的系数为)         |
|     🔴 尖峰: 1.8 | 🟠 高峰: 1.6 | 🔵 平段: 1.0 | 🟢 低谷: 0.4 | 🟣 深谷: 0.3 |
|     （具体系数值将根据当前选择的政策版本动态显示）                      |
|  2. **节假日政策**：春节、“五一”国际劳动节、国庆节期间，12:00-14:00执行深谷电价。 |
+-----------------------------------------------------------------------+
```

**手机横屏特殊优化**:
*   进入“沉浸模式”，隐藏顶部导航栏，最大化图表区域。
*   点击热力图任一色块后，底部弹出 `BottomSheet` 或 `Dialog` 显示该时段的详细信息和价格系数。

## 3. 竖屏模式 UI (Portrait View)

**适用场景**: **手机竖屏**

**核心策略**: **转置热力图 (Transposed Heatmap)**。将时间轴改为垂直方向，充分利用竖屏空间。

|                                                   |
|      1月 2月 3月 ... 7月 ... 12月 (X轴: 月份)     |

## 4. 后端 API 需求

*   `GET /api/v1/tou-rules/versions`: 获取可用版本列表。
*   `GET /api/v1/tou-rules/summary?version=...`: 获取指定版本的规则摘要及系数。
    *   返回数据需包含：版本号、系数表、以及按月份分组的 96 点时段数据。
