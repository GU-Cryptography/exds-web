### 3.2 新增功能详解

#### [新增] “待聚合”状态反馈
> **解决痛点**: 用户导入了 Excel，但忘了点“聚合”，导致报表中看不到最新数据，误以为导入失败。

1.  **全局提示 (Action Button Badge)**:
    *   在顶部的 `[⚡ 执行数据聚合]` 按钮上，增加醒目的徽标或文字后缀。
    *   示例: **"执行数据聚合 (🔴 125天次待处理)"**。
    *   逻辑: 统计系统中有 `raw_data` 但无对应 `unified_load` 的 (User, Date) 组合数量。

2.  **KPI 卡片**:
    *   新增/替换一张 KPI 卡片为 **“待聚合数据”**。
    *   显示: **涉及客户数 / 涉及总天数** (e.g., "12 户 / 125天")。
    *   颜色: **蓝色 (Info/Action)**，提示这是待办事项。

3.  **列表状态列**:
    *   在 Visual Gantt 条中，用 **`+` (加号或灰色阴影)** 表示“有Raw Data但未聚合”的区域。
    *   在“综合诊断”列，若存在未聚合数据，优先提示 **"� 有新数据待聚合"**。
