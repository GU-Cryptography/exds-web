# 趋势分析指标优化建议

基于电力现货交易的决策辅助需求，建议在后续的子页面（如“波动分析”、“储能套利”等）中增加以下 3 类更具实战指导意义的数据指标。

## 1. 储能与调节潜力 (最实用的套利指标)

目前“日前 vs 实时”的价差主要用于决定**在哪个月份/市场**买电。对于**储能**或**可调节负荷**（如工厂倒班），更关心的是**一天之内的价差**。

*   **建议指标**: **平均日内最大峰谷差 (Avg Daily Max Spread)**
*   **计算方式**: `平均(每日最高价 - 每日最低价)`
*   **展示文案示例**:
    > "日均最大峰谷差 **680 元/MWh**，套利空间 **[极大/一般/微薄]**。"
*   **决策价值**: 直接告诉用户，如果配置储能或调整生产时间，每天理论上每兆瓦时能节省（或赚取）多少收益。

## 2. 关键时段锚点 (最直观的操作指南)

帮助用户快速识别价格规律，无需人工分析复杂的曲线图。

*   **建议指标**: **高低价集中时段 (Price Anchors)**
*   **计算方式**: 统计选定时间范围内，每日最高价和最低价最常出现在哪个小时（众数）。
*   **展示文案示例**:
    > "最高价常发于 **19:00**，最低价常发于 **13:00**。"
*   **决策价值**: 直观指导用户何时停机避峰、何时满负荷生产或充电。

## 3. 极端风险预警 (风控指标)

平均值往往会掩盖极端情况，交易员需关注“爆雷”概率。

*   **建议指标**: **极端价格频次 (Extreme Price Frequency)**
*   **计算方式**: 统计价格超过特定阈值（如 >800元 或 <100元，或基于历史分位数）的时段占比。
*   **展示文案示例**:
    > "极端高价(>800元)占比 **3.5%**，需注意避险。"
*   **决策价值**: 提示市场风险稳定性。如果占比高，说明市场极不稳定，建议预留风险金或购买期权锁价。

## 4. 价格和价差日趋势深度挖掘 (进阶分析)

针对“价格和价差日趋势”页面，建议增加以下维度以支持更深层的风险与策略分析。

### 4.1 日内价格波动范围 (Price Range / Volatility Shadow)
*   **痛点**: VWAP（加权均价）掩盖了日内剧烈波动。均价相同的一天，可能是全天稳定，也可能是极端震荡，两者风险截然不同。
*   **建议**: 在“日均价格趋势”图中，在价格曲线背后增加一个**半透明的阴影区域**，范围覆盖每日的**最高价**和**最低价**。
*   **价值**: 直观展示“今日均价高”是全天高企还是单点拉高，识别潜在风险或机会。

### 4.2 价差稳定性/标准差 (Spread Stability)
*   **痛点**: “日均价差”无法反映波动风险。如果价差在正负之间剧烈跳动，套利风险极高。
*   **建议**: 增加一条“价差标准差”曲线，或者在价差柱状图上增加“误差棒”显示波动范围。
*   **价值**: 帮助用户识别套利机会是“稳健”还是“高风险”。

### 4.3 日历效应标记 (Calendar Effect Markers)
*   **痛点**: 周末和节假日电价通常深跌，在趋势图上形成周期性“深坑”，易误导用户判断市场走弱。
*   **建议**: 在 X 轴日期下方或图表背景中，用浅灰色竖条标记出**周末**和**节假日**。
*   **价值**: 辅助用户自动剔除“正常下跌”，准确判断市场真实走势。

### 4.4 移动平均线 (Moving Averages - MA7)
*   **痛点**: 每日价格跳动剧烈，难以看清长期方向。
*   **建议**: 增加 7日（对应一自然周）移动平均线。
*   **价值**: 过滤单日随机波动，展示真正的市场中期走势（上涨/下跌通道）。
