# 现货价格分析模块 - 重构规划方案

> 版本: v2.0
> 更新时间: 2025-11-23
> 设计理念: 从"指标展示"转向"决策支持"

---

## 一、模块重构目标

### 1.1 核心定位

**从"看数据"到"做决策"**

| 维度 | 旧定位 | 新定位 |
|------|--------|--------|
| 功能目标 | 展示价格指标和图表 | 支持交易决策和风险管理 |
| 用户价值 | 了解市场发生了什么 | 知道应该怎么做 |
| 输出形式 | 图表和数值 | 策略建议和预警信号 |

### 1.2 决策场景覆盖

| 时间尺度 | 决策场景 | 核心问题 |
|---------|---------|---------|
| **年度** | 年度采购策略 | 各月价格趋势？哪些月份风险高？ |
| **月度** | 月度持仓调整 | 本月市场特征？应加仓还是减仓？ |
| **周度** | 周内策略优化 | 周几容易高价？周末策略如何调整？ |
| **日内** | 日前报价和实时交易 | 哪些时段做多/做空？价差方向如何判断？ |

│ │ 425元/MWh │ │ +18元    │ │ 11,500MW │ │ 🟡 中等    │        │
│ │ ↑5% vs昨日│ │ 正差为主  │ │ ⚠️敏感区间 │ │ 建议观望   │        │
│ └───────────┘ └───────────┘ └───────────┘ └───────────┘        │
├─────────────────────────────────────────────────────────────────┤

│   日前 vs 实时 + 尖峰平谷背景色                                  │
├─────────────────────────────────────────────────────────────────┤
│ C. 时段财务速览表格                                             │
│   尖峰/峰/平/谷: VWAP_DA | VWAP_RT | 价差 | 策略建议             │
├─────────────────────────────────────────────────────────────────┤

1. **市场状态评估**：基于竞价空间、价差方向、波动性自动判断

3. **风险预警**：自动识别高价风险时段和低价机会时段

### 模块2：月度分析 (Monthly Analysis)

**定位**：支持月度持仓策略调整

#### Tab 1: 月度趋势分析

**核心图表**：

1. **月度价格趋势** (折线图)
   - X轴: 日期
   - Y轴: 日均VWAP (日前/实时)
   - 辅助线: 月均价、移动平均线

2. **月度价差趋势** (柱状图)
   - 正差(红)/负差(绿)
   - 统计: 正差天数占比、平均价差

3. **月度竞价空间趋势** (双Y轴)
   - Y1: 日均竞价空间
   - Y2: 日均价格
   - 用途: 验证"空间↑→价格↑"的关系

**核心指标卡片**：
- 月均VWAP_RT / VWAP_DA
- 月均价差及方向
- 平均竞价空间
- 高价天数 (VWAP > 月均+1σ)
- 低价天数 (VWAP < 月均-1σ)

#### Tab 2: 月度特征对比

**核心功能**：对比不同月份的市场特征

**分析维度**：

| 对比项 | 4月 | 5月 | 差异分析 |
|--------|-----|-----|---------|
| 日均价格区间 | 320-510 | 290-560 | 5月波动更大 |
| 月均价格 | 426 | 416 | 5月略低 |
| 正差天数占比 | 67% | 71% | 5月正差更多 |
| 竞价空间敏感度 | 中 | 高 | 5月更敏感 |
| 主导因子 | 负荷 | 负荷+光伏 | - |

**可视化**：
- 两月价格分布箱线图对比
- 两月分时价格曲线对比
- 两月竞价空间-价格散点图对比

#### Tab 3: 月度策略建议

**输出内容**：

```
┌─────────────────────────────────────────────────────────────────┐
│ 📋 本月市场特征总结                                              │
│ • 价格区间: 290-560元/MWh，波动幅度较大                          │
│ • 正差为主: 71%天数为正差，平均+18元                              │
│ • 敏感区间: 竞价空间10000-12000MW时价格波动最大                   │
│ • 主导因子: 白天受光伏偏差影响，夜间受负荷偏差影响                 │
├─────────────────────────────────────────────────────────────────┤
│ 📈 中长期持仓建议                                                │
│ • 竞价空间 < 11000MW: 减少持仓，风险敞口较高                      │
│ • 竞价空间 > 13000MW: 可适度加仓，价格有支撑                      │
│ • 关注指标: 日均竞价空间、光伏/负荷预测偏差                       │
├─────────────────────────────────────────────────────────────────┤
│ ⚠️ 风险提示                                                      │
│ • 本月价格对竞价空间高度敏感，需密切监控                          │
│ • 午后(12-15时)大正差概率高，谨慎做多日前                         │
│ • 降雨天气会显著抑制正差，关注气象预报                            │
└─────────────────────────────────────────────────────────────────┘
```

#### Tab 3: 价差归因分析

**核心图表**：

1. **价差主图** (96点柱状图)
   - 正差(红)/负差(绿)

2. **偏差归因** (多Y轴折线图)

- 白天价差主要由光伏偏差驱动
- 傍晚价差由负荷偏差主导

#### Tab 4: 异常价差深度归因 ⭐ 新增

**定位**：当日前和实时价格差异很大时（如|价差|>50元），进行多维度详细归因分析

**触发条件**：
- 时段价差绝对值 > 设定阈值（默认50元/MWh）
- 用户点击异常时段进行深度分析

**归因维度**：

| 系统负荷 | 18,500 MW | 19,200 MW | +700 MW | +3.8% | 高 | 正差↑ |
| 光伏出力 | 1,200 MW | 1,050 MW | -150 MW | -12.5% | 高 | 正差↑ |
| 风电出力 | 450 MW | 480 MW | +30 MW | +6.7% | 低 | 负差↑ |
| 火电出力 | 15,800 MW | 15,600 MW | -200 MW | -1.3% | 中 | 正差↑ |

2. **备用容量分析**

| 备用类型 | 日前计划 | 实时值 | 变化 | 影响分析 |
|---------|---------|--------|------|---------|
| 正备用 | 1,500 MW | 1,200 MW | -300 MW | 备用紧张，抬升价格 |

| 备用率 | 8.1% | 6.5% | -1.6% | 低于警戒线，风险较高 |

| 机组/线路 | 检修类型 | 容量影响 | 时段 | 对价差影响 |

## 八、总结

本次重构的核心理念是**从"看数据"到"做决策"**：

1. **多时间尺度覆盖**: 年/月/周/日全覆盖
2. **决策导向输出**: 每个模块输出明确的策略建议
3. **因子驱动分析**: 理解"为什么"，而非仅展示"是什么"
4. **风险预警机制**: 自动识别高风险时段和机会

预期效果：让交易员能快速回答：
- "今天市场状态如何？应该加仓还是减仓？"
- "哪些时段做多/做空？置信度和风险如何？"
- "本月市场有什么特征？需要调整策略吗？"
- "竞价空间处于什么水平？价格会如何变化？"
