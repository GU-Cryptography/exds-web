## 设计规范

### 颜色方案

| 用途 | 颜色 | Hex Code | 备注 |
|------|------|----------|------|
| 主色调 | 渐变蓝紫 | linear-gradient | 用于汇总面板背景 |
| 正价差/做多 | 红色 | #f44336 | 中国股市习惯 (涨/红) |
| 负价差/做空 | 绿色 | #4caf50 | 中国股市习惯 (跌/绿) |
| 警告/中等 | 橙色 | #ff9800 | |
| 中性/观望 | 灰色 | #9e9e9e | |

### 组件规范

#### 1. 汇总面板 (Summary Panel)
- **样式**: 渐变背景 (`#667eea` -> `#764ba2`), 白色文字
- **内容**: 
  - **指标速览**: 统计高确定性时段数、列出前三名、警示高风险时段数
  - **指标定义**: 解释"高确定性"和"一致性评分"的计算逻辑
- **圆角**: 8px
- **阴影**: `0 4px 20px rgba(102, 126, 234, 0.3)`

#### 2. 箱线图 (Box Plot)
- **实现方式**: Recharts `ComposedChart` + `ReferenceArea`(箱体) + `ReferenceLine`(须线)
- **高度**: 桌面端 400px, 移动端 350px
- **X轴**: 时段编号 0-47 (或1-48)
- **颜色逻辑**: 中位数 >= 0 显示红色(正价差)，< 0 显示绿色(负价差)
- **全屏功能**: 支持横屏全屏查看

#### 3. 数据表格
- **排序列**: 支持所有数值列排序
- **推荐指数**: 使用 1-5 格信号条展示 (`signal_strength`)
- **一致性**: 进度条显示 (>=70% 绿色, <70% 橙色)
- **Chip组件**: 用于 推荐策略、置信度、风险等级

## 交互规范

### 图表交互
- **全屏**: 点击全屏按钮 → 横屏模式显示
- **Tooltip**: 鼠标悬停显示详细数值 (Min, Q1, Median, Q3, Max)

### 表格交互
- **排序**: 点击列标题切换升序/降序
- **分页**: 默认每页25条, 支持切换 10/25/50
- **导出**: 支持导出当前数据为 Excel

## 技术实现要点

### 前端技术栈
- **UI框架**: Material-UI v7
- **图表库**: Recharts
- **全屏功能**: `useChartFullscreen` Hook
- **API**: `trendAnalysisApi.fetchTimeSlotStats`

### 关键组件
1. `TimeSlotAnalysisTab.tsx` - 包含所有逻辑的主组件
2. `useChartFullscreen` - 全屏逻辑复用

### 数据流
```
用户选择日期范围 (SpotTrendAnalysisPage)
  ↓
Props 传入 TimeSlotAnalysisTab (startDate, endDate)
  ↓
调用 API: /api/v1/trend-analysis/timeslot-stats
  ↓
后端聚合计算
  ↓
返回 JSON: { kpis, timeslot_stats, box_plot_data }
  ↓
前端渲染 Summary Panel, Chart, Table
```
