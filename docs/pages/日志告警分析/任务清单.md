### 1.2 API 路由层
- [ ] 创建 API 路由 (`webapp/api/v1_task_monitor.py`)
  - [ ] `GET /api/v1/tasks/summary` - 获取统计面板数据
  - [ ] `GET /api/v1/tasks/execution-logs` - 获取任务日志列表
  - [ ] `GET /api/v1/tasks/execution-logs/{task_id}` - 获取任务详情
  - [ ] `GET /api/v1/alerts` - 获取告警列表
  - [ ] `GET /api/v1/alerts/{alert_id}` - 获取告警详情
  - [ ] `POST /api/v1/alerts/{alert_id}/acknowledge` - 确认告警
  - [ ] `POST /api/v1/alerts/{alert_id}/resolve` - 解决告警
  - [ ] `POST /api/v1/alerts/{alert_id}/ignore` - 忽略告警
  - [ ] `GET /api/v1/tasks/commands` - 获取命令列表
  - [ ] `POST /api/v1/tasks/trigger` - 触发远程任务
- [ ] 在 `webapp/main.py` 中注册路由
