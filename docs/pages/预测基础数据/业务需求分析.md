# 预测基础数据 - 业务需求文档

## 1. 功能概述

预测基础数据是电力交易辅助决策系统中价格预测模块的一个基础支持模块，旨在为预测模型分析师提供多维度电力数据的概览查看和对比分析能力。系统通过表格形式展示交易中心发布的周信息、日信息和实际发力三类数据集，支持多数据项的96点曲线对比分析，帮助用户直观了解预测基础数据准确性情况。

## 2. 业务背景

### 2.1 数据来源
见docs\pages\预测基础数据\dataset_structures.md，所有数据都来自交易中心平台下载
- **周预测数据**: weekly_forecast 平台发布的下周预测数据
- **日前发布数据**: daily_release 平台每日发布的短期预测数据，包含负荷、新能源出力等预测
- **实时发力数据**: real_time_generation，real_time_tieline 每日发布的实际出力数据，作为事后分析的基准

### 2.2 业务价值
- 提供预测精度评估的直观工具
- 支持多时间维度的数据对比分析
- 为交易决策提供数据可视化支撑
- 帮助用户了解系统运行态势

## 3. 数据维度分析

### 3.1 时间维度（表格列标题）
- **历史10天**: D-10, D-9, D-8, D-7, D-6, D-5, D-4, D-3, D-2, D-1
- **当天**: D（基准日期）
- **未来3天**: D+1, D+2, D+3
- **总计**: 14个日期节点，提供完整的时间轴视角

### 3.2 数据类型（表格行标题）

#### 周预测数据（5项）
1. **周系统负荷预测** - 次周系统总负荷预测值
2. **周统调光伏预测** - 次周光伏发电预测值
3. **周统调风电预测** - 次周风电发电预测值
4. **周水电（含抽蓄）预测** - 次周水电发电预测值，包含抽水蓄能
5. **次周省间联络线可用容量** - 次周省间联络线的可用容量预测

#### 日前发布数据（5项）
6. **短期系统负荷预测** - 短期系统总负荷预测值（96点）
7. **短期光伏预测** - 短期光伏发电预测值（96点）
8. **短期风电预测** - 短期风电发电预测值（96点）
9. **非市场化机组出力预测** - 非市场化机组的出力预测（96点）
10. **联络线总计划** - 联络线交换电力的总计划（96点）

#### 实时发力数据（5项）
11. **实际全网总出力** - 实际的全网总发电出力（96点）
12. **实际风电出力** - 实际的风电发电出力（96点）
13. **实际光电出力** - 实际的光伏发电出力（96点）
14. **实际水电（含抽蓄）出力** -  实际的水电（含抽蓄）总出力（96点）
15. **联络线总计划值** - 实时的联络线总计划数据（96点）

## 4. 用户交互需求

### 4.1 日期导航功能
- **基准日期选择器**: 支持用户选择具体的基准日期（D日）
- **左右切换按钮**: 快速切换到前一天/后一天
- **自动更新**: 表格数据根据基准日期动态刷新

### 4.2 数据选择机制
- **多选支持**: 支持同时选择多个数据项进行对比
- **选择状态管理**: 清晰的选中/未选中状态显示
- **批量操作**: 提供全选、清空等批量操作功能

### 4.3 数据状态展示

#### 桌面端
- **有数据状态**: 显示"月-日 + 复选框"，月-日表示该数据项的日期
- **无数据状态**: 显示"无数据"文字，灰色显示，不可选择
- **已选中状态**: 复选框显示为选中状态，单元格浅蓝背景

#### 移动端
- **有数据状态**: 显示复选框（简化，不显示日期）
- **无数据状态**: 显示 `—` 符号，灰色显示，节省空间
- **已选中状态**: 复选框显示为选中状态

#### 表格图例说明
为帮助用户理解符号含义，在表格下方显示图例：
- `☑ 已选中` - 该数据项已被选择，曲线已显示在图表中
- `☐ 可选择` - 该数据项有数据，可点击选择
- `—  无数据` - 该日期该数据项无数据

### 4.4 操作功能
- **实时图表联动**: 勾选/取消勾选数据点时，下方图表自动更新，无需点击按钮
- **清空选择**: 重置所有复选框为未选中状态
- **图表全屏**: 支持将96点曲线图全屏放大查看

### 5.3 交互流程设计

#### 基准日期切换与相对位置保持
```
场景：用户选择了"周负荷"的D-7和D两个数据点

初始状态（基准日期=2025-01-15）：
- 已选：D-7(01-08)、D(01-15) → 对应曲线：01-08和01-15

用户操作：切换基准日期到2025-01-14

系统响应：
1. 基准日期更新为2025-01-14
2. 表格日期列更新：
   - D-10 → 01-04（原01-05）
   - D-7  → 01-07（原01-08）
   - D    → 01-14（原01-15）
   - D+2  → 01-16（原01-17）
3. 保持相对位置选择：D-7和D仍然选中
4. 自动加载新曲线：01-07和01-14的曲线
5. 图表更新显示新曲线

关键：使用相对偏移量（如"1_-7"）作为选择key，而非绝对日期
```

#### 数据选择与图表联动流程
```
用户操作                     系统响应
   │
   ├─► 页面初始加载             ──► 默认基准日期为前一日（D = yesterday）
   │                              默认数据类型为"周预测"
   │                              桌面端：固定13天范围（D-10到D+2）
   │                              移动端：默认"近3天"范围
   │                              预加载数据可用性（查询12:00时间点）
   │
   ├─► 切换数据类型Tab          ──► 切换到对应数据类型的表格（5项）
   │   (周预测/日预测/实际数据)     保持已选数据项的相对位置
   │
   ├─► 移动端切换日期范围       ──► 表格显示对应日期列（3/8/10列）
   │   (近3天/近7日/历史10日)       保持已选数据项的相对位置
   │
   ├─► 勾选某个数据点            ──► 图表立即显示该曲线
   │   (如D-7的周负荷)               底部显示"已选择 N 项数据"
   │
   ├─► 切换基准日期              ──► 1. 更新表格所有日期列
   │   (如从01-15改到01-14)         2. 保持相对位置选择（如D-7仍选中）
   │                              3. 自动加载新日期的曲线
   │                              4. 图表更新显示
   │
   ├─► 点击[清空]按钮           ──► 清空所有Tab的复选框

```

### 5.4 视觉设计规范

#### 表格样式
- **列标题**:
  - 第一行：相对日期（D-10, D-9, ..., D-1, D, D+1, D+2）
  - 第二行：绝对月-日（01-04, 01-05, ..., 01-14, 01-15, 01-16）
  - Sticky Header：滚动时始终可见
- **行标题**: 数据类型名称左对齐，字体加粗
- **复选框状态**:
  - ✅ 已选中: 蓝色复选框，单元格浅蓝背景
  - ☐ 未选中: 灰色复选框，白色背景
  - 无数据: 灰色"—"符号
- **单元格高度**: 固定 48px，确保一致性，避免切换日期时跳跃
- **单元格间距**:
  - 桌面端: `px: 2, py: 1`
  - 移动端: `px: 0.5, py: 0.5`

#### 图表样式
- **容器高度**:
  - 桌面端: `400px`
  - 移动端: `350px`
- **曲线颜色**: 使用预定义颜色数组，自动分配
- **图例格式**: `数据项名称(月-日)`，如"周系统负荷预测(01-14)"
- **空状态**: "请在上方表格中选择数据项进行对比分析"
- **全屏按钮**: 右上角，zIndex: 1400

## 6. 技术约束和要求

### 6.1 性能要求
- **响应时间**: 表格数据加载时间 < 2秒
- **图形渲染**: 96点曲线图渲染时间 < 1秒
- **并发支持**: 支持多用户同时访问

### 6.2 兼容性要求
- **浏览器**: 支持Chrome、Firefox、Safari、Edge等主流浏览器
- **移动端**: 适配手机和平板设备
- **分辨率**: 支持1366x768到1920x1080等常见分辨率

### 6.3 数据准确性要求
