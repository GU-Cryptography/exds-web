根据《江西省电力零售市场实施细则（试行 4.0 修订版）》及配套的结算示例说明，关于零售用户月度结算均价封顶的规定如下：

### 1. 封顶价格的定义与计算基准
零售套餐中的全电量封顶价格并非一个固定数值，而是基于当月批发市场的交易情况动态计算的。
*   **基准价格**：**当月市场化用户中长期交易平段合同加权平均价**。
    *   *具体定义*：即当月全部售电公司、批发用户的年度、多月、月度各类中长期交易平段合同的加权平均价。

### 2. 封顶系数与计算公式
封顶价格根据月份类型（尖峰月份与非尖峰月份）设定了不同的上浮比例（封顶系数）：

*   **非尖峰月份**：
    *   **上浮比例（k1）**：不超过 **10%**。
    *   **计算公式**：封顶价格 = 当月平段合同加权均价 × (1 + 10%)。
*   **尖峰月份**：
    *   **上浮比例（k2）**：不超过 **15%**。
    *   **计算公式**：封顶价格 = 当月平段合同加权均价 × (1 + 15%)。

### 3. 结算执行逻辑
*   **触发条件**：在进行月底结算时，系统会计算零售用户根据其签约套餐（如固定价+联动、价差分成等模式）算出的实际结算均价。
*   **执行规则**：
    *   若**实际结算均价 ≤ 封顶价格**：按实际结算均价执行。
    *   若**实际结算均价 > 封顶价格**：按**封顶价格**进行结算。

### 4. 举例说明
假设某月为**非尖峰月份**，当月全省市场化用户中长期平段合同加权平均价为 **454.16 元/兆瓦时**。
*   **封顶价格** = 454.16 × (1 + 10%) = **499.58 元/兆瓦时**。
*   **情景**：若某零售用户根据套餐算出来的结算价格为 **501.21 元/兆瓦时**。
*   **结果**：由于 501.21 > 499.58，该用户当月最终结算价将被强行压降至 **499.58 元/兆瓦时**。

这一机制旨在保护终端用户，防止因批发市场价格波动或不合理的套餐条款导致用户侧电费过高,,,。
