# 系统监控模块前端UI设计方案（精简版）

> **基于**: [全系统统一任务调度与监控方案.md](./全系统统一任务调度与监控方案.md)

## 1. 功能范围

仅包含三个核心功能：
1. **统计面板** - 三个子系统（Web/Forecast/RPA）的健康度概览
2. **告警列表** - 未处理告警展示与操作
3. **日志列表** - 任务执行日志查询

```
├── pages/
│   └── TaskMonitorPage.tsx          # [NEW] 主页面
└── components/
    └── task-monitor/                # [NEW] 组件目录
        ├── ServiceHealthCard.tsx    # 服务健康卡片

        └── TaskLogTable.tsx         # 日志表格
```

---

## 4. API 接口

| 接口 | 方法 | 描述 |
|------|------|------|
| `/api/v1/tasks/summary` | GET | 获取统计面板数据 |
| `/api/v1/alerts` | GET | 获取告警列表 |
| `/api/v1/alerts/{id}/status` | PUT | 更新告警状态 |
| `/api/v1/tasks/logs` | GET | 获取任务日志 |
