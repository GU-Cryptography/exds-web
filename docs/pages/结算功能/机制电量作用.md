根据最新的《江西省电力市场规则（试行 4.0 修订版）结算实施细则部分示例说明》，机制电量参与计算“标准值”（用于确定偏差回收费用）的逻辑比之前的理解更为复杂和深入。

简而言之，**机制电量在计算“签约比例”时被算作已签约电量，而在计算“标准值费用”时，它被“虚拟”为一份价格等于当日市场化均价（或用户自身均价）的中长期合约参与差价结算。**

以下是详细的运作机制：

### 1. 在计算“签约比例”时：计入分子
在判断售电公司是否满足 **80% ~ 120%** 的签约比例要求时，**机制电量是包含在分子里的**，而不是从分母中扣除。

*   **计算公式**：
    $$签约比例 = \frac{持有的市场化中长期合同量 + \mathbf{机制电量}}{实际总用电量}$$
*   **作用**：机制电量直接帮助售电公司提升签约比例，使其更容易达到 80% 的下限要求。

### 2. 在计算“标准值费用”时：被“虚拟”计价
这是最关键的差异点。虽然机制电量在**实际结算**中不产生差价（费用为0），但在**标准值（考核基准线）的计算模型**中，它被假设为**“产生差价”**。

根据不同的签约情况，机制电量在标准值计算中的身份不同：

#### A. 场景一：签约比例不足 80%（补足模式）
如果（市场化合约 + 机制电量） < 总用电量的 80%，系统会自动补足至 80%。
*   **实际情况**：机制电量不产生差价支出。
*   **标准值模型**：**机制电量被视为一份“价格 = 当日全市场用户侧中长期均价”的合约**。
*   **计算逻辑**：
    $$标准值差价支出 = Q_{市场化} \times (P_{合同} - P_{参考}) + (\mathbf{Q_{机制}} + Q_{补足缺口}) \times (\mathbf{P_{全市场均价}} - P_{参考})$$
*   **后果（偏差回收）**：
    *   如果当日**市场合约均价 > 现货价格**（即签合约比买现货贵）：标准值模型会认为机制电量“应该”产生高昂的差价支出。但你实际没付这笔钱（因为机制电量无差价）。
    *   **结果**：这部分被“省下来”的钱（$Q_{机制} \times (P_{均价} - P_{现货})$）会被算作偏差回收费用，**向售电公司回收**。

#### B. 场景二：签约比例超过 120%（削减模式）
如果（市场化合约 + 机制电量） > 总用电量的 120%，系统会按 120% 封顶计算。
*   **标准值模型**：系统直接按 **总用电量 × 120%** 作为虚拟合约量，并全部按**用户自身的市场化合约均价**计算差价。
*   **后果**：此时机制电量实际上被“同化”为了用户的高价/低价市场化合约，参与到 120% 的封顶计算中。

### 3. 举例说明（基于官方示例）

**假设条件**：
*   用户用电量：87 MWh
*   持有的市场化合约：40 MWh（价格 461.8元）
*   分配的机制电量：20 MWh（无差价）
*   现货参考价：436.8元
*   市场均价：466.5元

**步骤一：算比例**
$$比例 = \frac{40 + 20}{87} = 68.97\%$$
（低于 80%，触发补足机制）

**步骤二：算实际差价（你实际付的钱）**
$$实际差价 = 40 \times (461.8 - 436.8) + \mathbf{20 \times 0} = 1000 \text{元}$$
*(注：机制电量实际不产生差价)*

**步骤三：算标准值差价（你理论上该付的钱）**
标准模型假设你有 80% 的电量签了合约。
$$Q_{补足总额} = 87 \times 80\% - 40 (\text{市场化}) = 29.6 \text{MWh}$$
*(注意：这里把机制电量的那部分坑位，也统统按市场均价重算了)*
$$标准差价 = 40 \times (461.8 - 436.8) + \mathbf{29.6 \times (466.5 - 436.8)} = 1000 + 879 = 1879 \text{元}$$

**步骤四：算回收（多退少补）**
$$回收费用 = 标准值支出 (1879) - 实际支出 (1000) = 879 \text{元}$$

### 总结
在计算标准值时，**机制电量失去了“免责金牌”**。
*   它被当作一份**随行就市（按市场均价）**的虚拟合约。
*   如果市场上的合约很贵，而机制电量让你避开了这些成本，偏差回收机制会把这部分“避开的成本”计算出来并回收。
*   **核心逻辑**：防止用户利用机制电量在合约价格高企时“搭便车”获利（少付差价）。
